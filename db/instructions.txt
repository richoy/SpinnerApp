To add an admin user:

1: Send a post request to /users/sign up with the body:

{
    "username": "",
    "password": "",
    "firstname": "",
    "lastname": ""
}

2. Open database.
3. use SpinnerApp
4. db.users.find().pretty()  -> To check the users.
5. db.users.update({"username": ""}, {$set: {"admin":true}}) -> To set to admin user.
